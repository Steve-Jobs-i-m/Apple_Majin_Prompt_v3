# 2カラムテーブルスライド要件定義

## 目的
データを2カラムのテーブル形式で整理し、比較や一覧表示を行います。仕様比較や価格表に最適です。

## 対応画像
- **参照画像**: `Apple_like_template/Apple_like_template.009.jpeg`
- **画像説明**: 黒背景に中央揃えの2カラムテーブル、各カラムに見出しと数値行

## 既存スライド型
- **type**: `table`
- **実装関数**: `createTableSlide()` (`src/slides.js`)

## レイアウト仕様
- **スライド比率**: 16:9 (960 × 540pt 基準)
- **背景**: ブラック (`#000000`)
- **テーブルエリア**:
  - 位置: 中央揃え
  - 幅: 700pt～800pt
  - top: 150pt 付近
- **カラム数**: 2～4列
- **行間隔**: 16pt～24pt

## タイポグラフィ
- **ヘッダー**:
  - サイズ: 28pt～32pt
  - ウェイト: 700 (Bold)
  - カラー: `#FFFFFF`
- **セル内容**:
  - サイズ: 20pt～24pt
  - ウェイト: 400 (Regular)
  - カラー: `#E0E0E0`
  - 配置: CENTER

## カラーパレット
- **背景色**: `#000000`
- **ヘッダー背景**: `#2A2A2A` (オプション)
- **セルボーダー**: `#3A3A3A` (1pt)
- **テキスト**: `#FFFFFF`, `#E0E0E0`

## ビジュアル要素
- **区切り線**: 細いライン（1pt）でカラムと行を区切る

## データスキーマ例
```json
{
  "type": "table",
  "title": "Model Comparison",
  "headers": ["Specification", "Value"],
  "rows": [
    ["Display", "14.2-inch"],
    ["Processor", "M4 Pro"],
    ["Memory", "36GB"],
    ["Storage", "1TB SSD"]
  ],
  "notes": "製品仕様をテーブル形式で表示。"
}
```
